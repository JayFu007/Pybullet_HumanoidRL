# 基于Pybullet打造的机器人强化学习环境：Pybullet_HumanoidRL
本项目是一个基于Pybullet的强化学习环境，主要用于训练和测试人形机器人（Humanoid Robot）的强化学习算法。该项目的目标是为研究人员和开发者提供一个易于使用和扩展的框架，以便在Pybullet中进行人形机器人相关的强化学习实验。
